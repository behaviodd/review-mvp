## 개요

리뷰 템플릿은 사이클에서 사용할 질문지의 청사진입니다. 섹션과 질문을 구성해두면 여러 사이클에서 재사용할 수 있습니다.

## 단계별 안내

### 1) 템플릿 목록 열기

좌측 메뉴 **리뷰 운영 › 템플릿** 으로 이동합니다.

![템플릿 목록 화면](/guide-images/review-cycle/01-template-list.png)

### 2) 새 템플릿 만들기

오른쪽 상단 **새 템플릿** 버튼을 누르면 빈 템플릿 빌더가 열립니다.

![템플릿 빌더 진입 화면](/guide-images/review-cycle/02-template-builder-empty.png)

### 3) 섹션과 질문 추가

- **섹션 추가** — 같은 주제의 질문들을 묶는 그룹입니다. 예: "성과", "역량", "협업".
- 각 섹션 안에 **질문 추가** 로 객관식·주관식·점수형 질문을 만듭니다.
- 드래그로 순서를 바꿀 수 있습니다.

![섹션과 질문 구성 화면](/guide-images/review-cycle/03-template-sections.png)

### 4) 미리보기로 확인

상단 **미리보기** 버튼으로 작성자가 보게 될 화면을 확인합니다.

### 5) 저장

**저장** 을 누르면 템플릿 목록으로 돌아갑니다. 새 템플릿이 목록 상단에 표시됩니다.

## 자주 묻는 질문

**Q. 템플릿을 수정하면 이전 사이클에도 영향이 있나요?**
A. 아니요. 사이클은 발행 시점의 템플릿을 **스냅샷** 으로 고정합니다. 이후 템플릿을 수정해도 진행 중인 사이클의 질문은 그대로 유지됩니다.

**Q. 기본 템플릿이 뭔가요?**
A. 시스템에서 제공하는 시드 템플릿입니다. 목록 최상단에 항상 표시되고, 그대로 쓰거나 복제 후 수정할 수 있습니다.
