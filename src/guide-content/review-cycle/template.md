## 개요

리뷰 템플릿은 사이클에서 사용할 질문지의 청사진입니다. 섹션과 질문을 미리 구성해두면 여러 사이클에서 재사용할 수 있습니다.

발행된 사이클은 **발행 시점의 템플릿을 스냅샷으로 동결**하므로, 이후 템플릿을 수정해도 진행 중인 사이클의 질문은 유지됩니다.

## 단계별 안내

### 1) 템플릿 목록 열기

좌측 메뉴 **리뷰 운영 › 템플릿** 으로 이동합니다.

목록은 한 페이지에 최대 15개까지 표시됩니다. 그 이상은 페이지네이션으로 넘기거나, 상단 검색창으로 이름·설명을 검색할 수 있습니다.

![템플릿 목록 화면](/guide-images/review-cycle/01-template-list.png)

### 2) 새 템플릿 만들기

오른쪽 상단 **새 템플릿** 버튼을 누르면 빈 템플릿 빌더가 열립니다.

처음 진입 시 빈 섹션 1개가 자동으로 추가되고 편집 모드로 들어갑니다. 섹션 이름을 비워두면 저장이 막히니 한 글자라도 입력해주세요.

![템플릿 빌더 진입 화면](/guide-images/review-cycle/02-template-builder-empty.png)

### 3) 섹션과 질문 추가

- **섹션** — 같은 주제의 질문들을 묶는 그룹입니다. 예: "성과", "역량", "협업".
- **질문** — 각 섹션 안에 객관식·주관식·점수형으로 추가합니다.
- 드래그로 섹션과 질문 순서를 바꿀 수 있습니다.

![섹션과 질문 구성 화면](/guide-images/review-cycle/03-template-sections.png)

### 4) 미리보기

상단 **미리보기** 버튼으로 작성자가 보게 될 화면을 확인합니다.

### 5) 저장

**저장** 을 누르면 템플릿 목록으로 돌아갑니다. 새로 만든 템플릿은 목록 상단(기본 템플릿 바로 아래)에 표시됩니다.

## 자주 묻는 질문

**Q. 기본 템플릿이 뭔가요?**
A. 시스템에서 제공하는 시드 템플릿입니다. 목록 최상단에 항상 고정되고, 그대로 쓰거나 복제 후 수정할 수 있습니다.

**Q. 템플릿을 수정하면 이미 발행된 사이클에도 영향이 있나요?**
A. 아니요. 발행 시점의 템플릿이 사이클에 동결되므로, 이후 템플릿 수정은 새로 발행하는 사이클부터만 반영됩니다.

**Q. 템플릿을 삭제할 수 있나요?**
A. 진행 중·완료된 사이클이 사용 중인 템플릿은 삭제가 막힙니다. 사용 이력이 없으면 삭제 가능합니다.
