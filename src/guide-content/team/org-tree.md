## 개요

조직은 자유 트리 구조로 구성됩니다. 회사 → 부서 → 팀 → 분과 등 깊이 제한 없이 만들 수 있습니다. 조직 단위로 평가 대상을 정하거나 조직장 권한을 부여하는 데 사용됩니다.

## 단계별 안내

### 1) 조직도 패널 열기

**구성원 관리 › 구성원** 페이지 우측에 조직도 패널이 있습니다. 트리 구조로 회사 전체 조직이 표시됩니다.

![구성원 페이지의 조직도 패널](/guide-images/team/70-org-panel.png)

### 2) 조직 추가

조직 옆 **+** 아이콘을 누르면 하위 조직을 추가합니다. 루트 옆에 추가하면 새 최상위 조직이 됩니다.

![조직 추가 입력 화면](/guide-images/team/71-org-add.png)

이름을 입력하고 Enter 또는 ✓ 클릭으로 확정합니다.

### 3) 이름 변경

조직 이름을 더블클릭하거나 우클릭 메뉴에서 **이름 변경** 을 선택해 편집 모드로 전환합니다.

### 4) 순서·계층 변경 (DnD)

조직 노드를 드래그해서 순서를 바꾸거나 다른 조직 안으로 이동시킬 수 있습니다:

- **위/아래 인접 영역** — 같은 부모 안에서 순서만 바뀜
- **노드 안 영역** — 해당 조직의 하위로 이동

![DnD 동작 — 드롭 인디케이터 표시](/guide-images/team/72-org-dnd.png)

조직장(headId)이 지정된 조직은 트리 상에서 **사람 아이콘 + 이름** 으로 표시됩니다.

### 5) 조직장 지정

조직 노드 우측 메뉴에서 **편집** 을 선택하면 조직 편집 다이얼로그가 열립니다. **조직장** 드롭다운에서 활성 멤버 중 한 명을 선택합니다.

![조직장 지정 화면 — 조직 편집 다이얼로그의 조직장 드롭다운](/guide-images/team/73-org-head.png)

조직장으로 지정된 사용자는 다음 권한을 자동으로 갖습니다:
- 해당 조직 + 하위 조직 멤버에 대한 **하향 평가**(downward) 작성
- 해당 조직 멤버의 **승인 대기** 처리 (조직장이 그룹 멤버일 때)

### 6) 조직 삭제

조직 우클릭 메뉴 **삭제**. 삭제 시:
- 하위 조직이 있으면 차단됩니다 (먼저 하위를 비우거나 삭제해야 함)
- 소속 멤버가 있으면 경고 후 멤버를 다른 조직으로 이동시키도록 안내합니다

## 자주 묻는 질문

**Q. 조직 이름을 바꾸면 과거 리뷰의 부서 정보도 바뀌나요?**
A. 사이클이 `snapshot` 모드면 발행 시점의 조직 이름이 동결되어 영향받지 않습니다. `live` 모드면 항상 현재 이름이 표시됩니다. 자세한 운영 정책은 운영자에게 확인하세요.

**Q. 조직장이 admin role 사용자여도 되나요?**
A. 네. admin 도 조직장 + 평가권자가 될 수 있습니다.

**Q. 한 사람이 여러 조직의 조직장을 맡을 수 있나요?**
A. 네. 같은 사용자를 여러 조직의 headId 로 지정할 수 있습니다.
